# Games Collection

A collection of fun two-player browser games. No installation needed — just click and play!

---

## Tic Tac Toe

Classic two-player Tic Tac Toe with custom player names.

**Features:**
- Custom player names
- Win detection (rows, columns, diagonals)
- Tie detection
- Reset button

**Play now:** [kiranv0809.github.io/game/](https://kiranv0809.github.io/game/)

---

## Connect Four

Drop pieces into a 6x7 grid and connect 4 in a row to win!

**Features:**
- Custom player names (Red vs Yellow)
- Win detection (horizontal, vertical, diagonal)
- Tie detection
- Play Again button

**Play now:** [kiranv0809.github.io/game/connectfour.html](https://kiranv0809.github.io/game/connectfour.html)
